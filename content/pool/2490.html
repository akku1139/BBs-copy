お久しぶりです。今回はlinuxのVMです。
まずhttps://console.cloud.google.com/を開き、必要なプロンプトに従ってログインしてください。
その後右上のターミナルアイコン[>_]をクリックします。

下の画像のような画面が出たら下記のコードを入力しEnterを押します。

  docker run -p 6070:80 dorowu/ubuntu-desktop-lxde-vnc or if you dont want root docker run -p 6080:80 -e USER=doro -e PASSWORD=password -v /dev/shm:/dev/shm dorowu/ubuntu-desktop-lxde-vnc

読み込みが終わったら”ウェブでプレビュー”をクリックし、ポートを変更を押します

ポートを6070に変更します。

その後、ウェブでプレビューをもう一回押すと…………..

以上です。
