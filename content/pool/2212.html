このやり方はGoogleアカウント,Githubアカウントが必要です。予め作成してください。
Githubアカウント→作成方法

最初にhttps://console.cloud.google.com/に入り、必要なプロンプトに従ってログインしてください。
その後右上のターミナルアイコン[>_]をクリックします。

下の画像のような画面が出たら下記のコードを入力しEnterを押します。

git clone https://github.com/NebulaServices/Nebula
cd Nebula
npm i
npm ci
PORT=3001 node .

ターミナルに http://localhost:3001/ というメッセージが表示されたら、それを cntrl + クリックしてください。
下の画像ようなのが開いたらひとまず成功です。

ここからはGitPod の手順です。
その後https://github.com/ で GitHub にサインアップします。
そしてhttps://github.com/NebulaServices/Nebulaにアクセスします。
右上のForkをクリックし、Forkを作成します。

Forkを作成し終わったら、https://gitpod.io/にアクセスします。
アクセスしたら “Continue with Giuhub“を選択し、予め作成していたGithubアカウントでログインしてください。

このようになったら、”New Workspace“をクリックしてください。
そしてnebulaと検索します。
出てきたものをクリック。読み込みが終わるまで待ちます。

この画面になったら”PORTS”をクリック。
下のように3000の列にカーソルを合わせ、右クリックし、”Make Public“をクリックします。
そのご右のアドレスをクリックして、Proxyが使えます！！

今回の方法は少しややこしい部分があるのでわからなければコメント欄で聞いてください。
