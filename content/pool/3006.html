  • chrome-extension://ここに自分のフィルタリングのID/manifest.json を開きます

フィルタリングのIDがわからない場合は、 Document: [メモ] Extension ID List

  • そのタブで、chrome://hang を開きます
    > ずっと読み込み中状態になってるならokです
  • タブ名を右クリックで「タブを複製」をクリックして複製します

複製したタブと複製前のタブは残しておく

  • chrome://extensions/?id=ここに自分のフィルタリングのID を新しいタブで開きます

フィルタリングのIDがわからない場合は、 Document: [メモ] Extension ID List

  • 下にスクロールし「Chrome ウェブストアのページに移動」をクリックし、
    開いた直後すぐに、escキー(左上)を押す。(無題って表記であればok)
  • 無題のタブで以下のbookmarkletを実行
javascript:(async () => await opener.chrome.developerPrivate.getExtensionInfo(new URLSearchParams(new URL(opener.location.href).search).get("id")).then(info => opener.chrome.developerPrivate.updateExtensionConfiguration({extensionId: info.id, fileAccess: !info.fileAccess.isActive})))()
  • そうすると複製前の方のタブが消え、複製したタブが残りますのでこのまま複製したタブはのこしておいてください
    (その他のchrome://extesionsや無題のページは消してどうぞ)

複製したタブが残ってないとフィルタリングが再度起動してしまいます
