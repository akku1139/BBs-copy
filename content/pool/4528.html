今回は、学校のChromebookなどで、Youtubeの見れるようにしたサイトを作りました。

できること

  • Youtube検索
  • 動画の再生
  • 動画のダウンロード

などなど…..

機能は追加する予定です。

閲覧方法

まず、

https://youtube.psannetwork.net (11/30 14:42時点確認 – ダウン中)

https://youtube.mq.gy/ (11/30 14:42時点確認 – ダウン中)

にアクセスします。(自宅サバなので停止している可能性あり)

検索欄に好きなものを入力して検索できます。

自分でホストする方法

今のところサーバーがまだ安定していないので、自分でサーバーを立てると比較的高速に閲覧できます。

必要なパッケージ

nodejsの最新バージョン

pm2

nodejs pm2 のインストール方法

Debian系の場合

sudo apt install curl
curl -Ls https://raw.githubusercontent.com/hirotomoki12345/chrome/main/chroot/setup_nodejs.sh -o setup_nodejs.sh
bash setup_nodejs.sh
npm install -g pm2
ホスト方法

ターミナルで、

curl -Ls https://raw.githubusercontent.com/hirotomoki12345/youtube-searcher/main/set.sh -o set.sh

を入力します。

次に、

bash ./set.sh

を実行します。

こんな画面が出てくるので、1 をおします。

しばらくすると、セットアップが完了します。

http://localhost:3503

にアクセスすると、サイトが表示されます。
