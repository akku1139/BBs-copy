こんにちはPsanです。

今回は、Web上で動作するChromeOSのエミュレーターを紹介します。

このプロジェクトは、MercuryWorkshopという開発グループのものです。

ソースコード
https://github.com/MercuryWorkshop/anuraOS

AnuraOSとは？

Webサイト上でエミュレートできる、Chromium OSに似た仮想Linux環境です
Google Chrome, Firefox, Safariで動作します。

実際に動か

「AnuraOS」の新しいバージョンが、もうすぐリリースされる予定だそうです。
ベータ版は、こちら: https://beta.anura.pro/

まず、https://anura.pro/　にアクセスします。

次にNextを押します。

LinuxなどのRootfsを入れて、Linuxのコマンドが使いたい人は、

Alphin Debian Arch　の3つが用意されています。

Linuxを使わない人は、Normal User を選択してください。

また、ダウンロードして実行しない場合は、No Download　を選択してください。

すると、セットアップが開始されるので、しばらく待つと、AnuraOSが起動します。

自分でAnuraOSをホストする Ubuntu

sudo apt install -y nodejs npm wget default-jdk inotify-tools rustc cargo make gcc docker.io
rustup target add wasm32-unknown-unknown

リポジトリのクローン

git clone --recursive https://github.com/MercuryWorkshop/anuraOS

makeの実行

cd anuraOS
make all

Rootfsを作る場合

make rootfs

選択肢が出てきたら、4を押します。

起動する

make server

http://localhost:8000　にアクセスします。
