～～最低条件～～
・デベロッパーモードであることが必要です。
・USB3.0以上のUSBと、32GB以上のストレージが必要です
・もう1台パソコンが必要です。

あとくそめんどくさいです

手順

自分のパソコンで、「Disk Management」を開きます。
C;を右クリックし、縮小を選択します。
パーティションを8192MB縮小します。
これで、8GBの空き容量ができます。
空き領域を右クリックし、「新しいボリューム」を選択します。
FAT32としてフォーマットし、Installerと名付ける。
8192MBのストレージを与える。

次にhttps://www.linuxmint.com/edition.php?id=299から「Torrent Download」をクリックします。
64-Bit “をクリックし、BitTorrentクライアントでファイルを開きます。
（お持ちでない方は、こちらからダウンロードしてください。 https://www.qbittorrent.org/download.php）
.isoファイルを入手したら、7-Zipで開いてください。
（7-Zipはこちらからダウンロードできます。 https://www.7-zip.org/download.html）
.isoファイルの内容をすべて、作成した新しいドライブパーティションにコピーします。
コンピュータを再起動し、Windowsを起動します。

その後、ログイン画面で、右下の電源ボタンを押し、shiftを押しながら、再起動を選択します。
shiftを押したままにしておきます。
ブルースクリーンがポップアップ表示されます。Use a Device を選択します。
Linux Mint、または UEFI が Linux Mint と呼ぶことに決めたものを選択し ます。
（Acer コンピュータの場合、”Linpus Lite “と表示されるかもしれません。）
用意してあるUSB を接続します。
デスクトップが表示されるので、”Install Linux Mint “をクリックします。
インストーラが “Installation Type” を聞いてくるので “Something Else” を選択します。
Device for Bootloader Installation “をクリックし、USBドライブの名前を選択します。USBドライブの名前の左側にあるテキストに注目してください。
これがパスです。（通常、”/dev/sda “と表示されます。）
インストーラーのページを、USBの名前の横に表示されていたパスが表示されるまでスクロールしてください。
USB上の既存のパーティション（/dev/sda1、/dev/sda2）を選択し、「-」を押して空き領域を作ります。
新しい空き領域を選択し、「+」を押して新しいパーティションを作成します。
サイズ500MB、パーティションの種類を作成します： Primary, Location for the Partition： この新しいスペースの始まりです。として使用する」というボックスを選択し、「EFIシステムパーティション」を選択します。OKを押す。
残りの空き領域を選択し、もう一度＋を押します。サイズはできるだけ大きく、Type of partition： Primary, Location for the partition： この新しいスペースのはじまり。Use as:」というボックスを選択し、「Ext4 journaling system」を選択します。
マウントポイント」と書かれたテキストボックスをクリックし、フォワードスラッシュを入力します。(/) OKを押します。

すべての準備が整ったら、USBがブートローダーのインストール用に設定されていることをもう一度確認し、「今すぐインストール」を押します。
タイムゾーンを選択します。
ユーザーを作成します。Encrypt my Home Directory（ホームディレクトリを暗号化する）」を必ず選択してください！これはセキュリティ上非常に重要なことです！
インストールが完了するのを待つ間に、ステップ3を実行します。
ステップ3：ファームウェア
Chromebookの電源を入れ、CTRL + ALT + Tを押してCroshを開きます。
必ず「shell」と入力して、bashに入るようにしてください。
cd; curl -LO mrchromebox.tech/firmware-util.sh && sudo bash firmware-util.sh 必ずChronosとして実行してください。rootで実行すると、間違った場所にインストールされ、物事を台無しにします。
読み込みが終わったら、1.を押してください。
完了するまで待ち、Chromebookを再起動してください。
Linux Mintのインストールが終了するのを待ってから、次に進みます。
USBをChromebookに挿入してください。
OS Verification画面でCTRL + Lを押し、1を押します。
画面下にプロンプトが表示されたらescapeを押し、Boot Menuを選択します。
USBを選択し、Enterキーを押します。
そしたら完了です。
